# Access Review Process

## Purpose

This process documents how user access should be reviewed periodically.

## Review Steps

1. Export user access list.
2. Send access list to manager or application owner.
3. Review whether each user still needs access.
4. Approve, remove, or investigate access.
5. Revoke unnecessary access.
6. Document review results.

## Security Goal

Access reviews reduce excessive permissions, privilege creep, and orphaned access.
